# Cozy Time Dashboard

> This project is a work in progress.  
> A cozy, minimalistic time dashboard with multiple time modes (Countdown, Stopwatch, Clock) and ambient background animations.

## Features (in progress)
- Countdown
- Stopwatch
- Clock (real time)
- Animation selector (Snow, Floating balls, Particles)
- Future: Pomodoro mode

## Note
This project is currently under development. Early commits may be messy as the code is gradually being refactored and improved.
